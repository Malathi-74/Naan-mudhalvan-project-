# Text-to-Speech Converter

A web-based Text-to-Speech (TTS) application built with **Flask** and **pyttsx3**, allowing users to convert any text into natural-sounding speech and download it as an MP3 file. Includes live preview using the browser's Web Speech API.

## Features
- Convert text to speech in **male** or **female** voice.
- Adjust **speech rate** dynamically.
- **Preview** speech in the browser before downloading.
- Download generated speech as **MP3**.
- Handles errors gracefully with detailed feedback.


## Technologies Used
- **Backend:** Python, Flask, pyttsx3
- **Frontend:** HTML, CSS, JavaScript, Font Awesome
- **Other:** Flask-CORS for cross-origin requests
